# AI-Powered Email-Centric Communication Platform

This project integrates email and multi-channel messaging into a unified communication management platform.  
It uses AI to summarize, analyze, and assist in replying to messages.

## Tech Stack
- Frontend: Next.js (React + TypeScript)
- Backend: Next.js API routes + MongoDB + Redis
- AI: Python FastAPI service
- Infra: Docker Compose (multi-service orchestration)
